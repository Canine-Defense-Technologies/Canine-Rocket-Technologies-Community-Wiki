{% hint style="info" %}
This is our internally researched and developed reference

### FDM

| Interface Medium | Tight | Loose |
| ---------------- | ----- | ----- |
| Enclosure        | 0.5   | 0.75  |
| Axle/Joint       | 0.25  | 0.5   |

| Interface Medium | Tight | Loose |
| ---------------- | ----- | ----- |
